But we have one more thing to do:

Separating JS, HTML, and CSS

We did the one-file version but not the 3 file one.

In a bigger projects, you typically don't want to put everything in a single HTML, especially when you're using the same styling format or script on multiple pages.

That will cause redundancy and more headaches in the long run.

To update our code, create two more files: script.js and style.css.

script.js contains what's in our script tag while style.css contains the styling we did.
